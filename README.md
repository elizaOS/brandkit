## üìê Usage Guidelines

### Logo Clear Space
Maintain a clear space around the logo equal to the height of the "E" in ElizaOS.

### Minimum Size
- Digital: 120px width minimum
- Print: 1 inch width minimum

### Do's
- ‚úÖ Use provided color variations
- ‚úÖ Maintain aspect ratios
- ‚úÖ Use appropriate format for medium (RGB for digital, CMYK for print)
- ‚úÖ Ensure sufficient contrast with backgrounds

### Don'ts
- ‚ùå Don't alter colors or gradients
- ‚ùå Don't distort or stretch logos
- ‚ùå Don't add effects (shadows, outlines, etc.)
- ‚ùå Don't recreate logos

## üì¶ File Format Guide

### When to Use Each Format

**PNG** - Best for:
- Web graphics with transparency
- Social media
- Email signatures
- Digital presentations

**SVG** - Best for:
- Responsive websites
- Scalable applications
- Animation
- Icons

**AI** - Best for:
- Professional print design
- Large format printing
- Design modifications (with approval)

**EPS** - Best for:
- Spot color printing
- Professional printing services
- Legacy systems
